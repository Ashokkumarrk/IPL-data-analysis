# 🏏 IPL Data Analysis Project

## 📌 Project Overview

This project involves *exploratory data analysis (EDA)* on IPL (Indian Premier League) match datasets. Using Python and powerful libraries like Pandas, Matplotlib, and Seaborn, we uncover interesting insights about teams, players, match outcomes, venues, and more.

## 🛠 Tools & Technologies Used

- *Python*
- *NumPy* and *Pandas* for data manipulation
- *Matplotlib* and *Seaborn* for data visualization
- *Jupyter Notebook* for development

## 📂 Datasets Used

- matches.csv: Match-level data
- players_country.csv: Player nationalities

## 📊 Analysis Performed

- Total matches per team
- Toss winner vs match winner
- Team-wise color-coded bar charts
- Most frequent Player of the Match
- Top Indian MoM players
- Super Over by year
- Venue analysis
- MoM by team
- Umpire stats

## 💡 Key Insights

- Certain venues host more matches
- Toss win ≠ match win
- Indian players dominate MoM list
- Super Overs increasing in recent years
